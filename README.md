# Project Sync (prj-sync) CLI

command line script to copy and sync environments configuration and local data of development projects


### Install
```
npm install -g .
```

### Usage

```
prj-sync
```

### Uninstall
```
npm uninstall -g prj-sync
```

### Run dev mode

```
npm run dev

pnpm dev

yarn dev
```


### Run tests

```
npm run test

pnpm test

yarn test
```
